# CodeAlpha Event Registration System

## Overview

A Django-based Event Registration System that allows users to view events, register for events, and manage registrations.

## Features

- View upcoming events
- View event details
- Register for events
- Store registration details
- View all registrations
- Cancel registrations
- Admin panel management
- Bootstrap styled interface

## Technologies Used

- Python
- Django
- SQLite
- HTML
- Bootstrap

## How to Run

1. Activate virtual environment
venv\Scripts\activate


2. Install requirements


pip install -r requirements.txt


3. Run migrations


python manage.py migrate


4. Start server


python manage.py runserver


Open:


http://127.0.0.1:8000/
